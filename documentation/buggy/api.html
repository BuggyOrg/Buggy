Buggy : API

Buggy API

To use buggy simply include "buggy.js" into your project or webpage. It will define the global nameBuggywhose public API is

Buggy = { create, Compose, Group, Graph, Resolve, Semantics }

Capitalized names represent Objects and lower case names are functions. The next part describes the API in more detail.

Buggy.create(config)

Compose

This contains all functions regarding the composition of programs.

Compose.compose(semantics, options)

The compose function creates a program using the given semantics and a set of options sepcifying the composition procedure.

semantics
Semantics must be a valid semantic database.
options
The options field describes details of the composition procedure. The field can be used for user defined parameters and thus it is not possible to list all values here, but it follows a certain structure and a few items are standard items

Compose.create-dependency-graph(semantics, options)

This function creates the dependency graph for the given semantics. On default it doesn't process any construction rules like postprocessing etc..

semantics
Seantics must be a valid semantic database.
options
The options field has the same parameters as the option field in Compose.compose plus the options postprocessing that determines if any postprocessing should be done
