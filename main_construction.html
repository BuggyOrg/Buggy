Semantic Programming

Natural Abstraction

Unlike other programming techniques, semantic programming creates abstraction layer naturally without limiting possible implementations.
With Buggy it is possible to:

  • program on an high abstraction level
  • exchange implementations easily
  • avoid any overhead associated with abstractions

Independence

Buggy is a based on data. It is thus possible to create many different input languages that describe the problem in an appropiate way, even graphically.

With that we create programs that can run in the browser or on any platform. We can generate source code or compile native code and decide to run the program on multiple machines or on a single core.

Buggy can translate every program into any programming language if a few minimal instructions are defined.
